# :woman_technologist: Developers Team

This project carried out at the IT Academy aims to learn to collaborate as a team and acquire knowledge in Git Flow (using Kanban board in Git) and the creation and integration of the database model connecting it with the application for the first time without using a framework.

#### This project it has been carried out by:

- [@ariamdev](https://github.com/ariamdev)
- [@MartaPunsola](https://github.com/MartaPunsola)
- [@Patrifr](https://github.com/Patrifr)


 Escape Room:
-

We need to develop an app to manage a virtual Escape Room, where users can enjoy exciting adventures and solve challenging enigmas. At level 1, as MySQL persistence.

The application must meet the following requirements:

- In our virtual Escape Room, which has a specific name, we offer a variety of themed rooms, intriguing tracks and unique decorative objects.
- Each room is assigned a level of difficulty to provide a balanced and challenging experience
- The tracks are designed with specific themes to guide players in finding solutions
- Decoration objects contribute to creating an immersive and memorable atmosphere in each room, using different types of materials.
- All elements have an associated price that reflects their value in the game
- Our virtual Escape Room must maintain an updated inventory of all the rooms, tracks and decoration objects available.
- In addition, we will keep a record of the total value of the inventory to have a clear vision of our assets.
- The application will offer a functionality to issue certificates of overcoming riddles, where the achievements achieved by the players during their experience in the Escape Room will be recorded.
- In addition, they may be granted possible gifts or rewards in recognition of their ability and skill in solving the challenges posed.

As for the features to be displayed on screen, it is expected to include at least the following:

- Create a new virtual Escape Room.
- Add a new room with its respective level of difficulty.
- Incorporate themed tracks to enrich the game experience.
- Introduce decorative objects to set the rooms in a unique way.
- Show the updated inventory, showing the available quantities of each element (rooms, tracks and decoration objects).
- View the total value in euros of the inventory of the Virtual Escape Room.
- Allow the withdrawal of rooms, tracks or decorative objects from the inventory.
- Generate sales tickets for different players.
- Calculate and show the total income generated by ticket sales of the Escape Room virtual.
- Notify users about important events in the Escape Room, such as adding new tracks, ending a room, etc.
- Users interested in these events may request to register in order to receive notifications when relevant events occur.
